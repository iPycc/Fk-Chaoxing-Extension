# 更新日志

## \[1.2.2] - 2026-05-29

### 新增

- 引入 `frontend-design` 极简视觉规范，全面重构 Popup 与 Modal 等 UI 组件，抛弃冗余装饰。
- 新增：进入页面自动收集并在扩展图标（Badge）展示真实题目数量功能。
- 新增：插件软关闭（开关）功能，可临时停用插件脚本加载。
- 增强：深度 iframe 题目拦截与异步重试机制，解决了部分页面初次加载抓取不到题目的问题。
- 重写 extractors\HomeworkExtractor.js 作业作答 提取器 支持异步处理 并支持新版DOM class `.questionLi` + `h3.mark_name`

### 优化

- 优化：AI 悬浮窗面板移除所有 Emoji 并替换为现代极简风格。
- 优化：Modal 弹窗样式更加扁平克制，增加窗口与内容区域的最大显示尺寸，方便长文本阅读。

## \[1.2.1] - 2026-01-07

### 新增

- README 开发者视角重写，新增架构图与核心流程，方便二次贡献

- 日志输出继续瘦身：AI 答题过程仅保留“开始/完成”两行，去除中间进度刷屏

### 修复

- 考试页面 AI 按钮初始化失败（`notify.js` 误用 `isMainPage` 判断导致提前返回）
- 重复初始化日志：顶层帧以外禁止输出「插件加载完成」等通用提示，减少控制台噪音

## \[1.2.0] - 2026-01-06

### 新增

- 全新黑白极简 Popup，集成「获取题目」「AI 答题」「重载扩展」与实时日志
- 跨 iframe 日志聚合：关闭再开 Popup 仍可查看历史记录
- 页面类型自动识别（作业 / 考试 / 其他）并高亮图标

### 优化

- 模块化拆分：`extractors` + `services` + `ui` + `ai-answer`，消灭“大泥球”脚本
- 统一日志格式 `[Fk-Chaoxing] 级别 信息`，支持 `debug/info/success/warning/error`
- 题目提取与 AI 答题共用 `QuestionCollector`，代码复用率↑

### 修复

- 修复点击「获取所有题目」时每个 iframe 都弹窗的 Bug（仅 `top` 帧响应消息）
- 修复 AI 答题在考试页面无法使用（移除 `isMainPage` 限制）
