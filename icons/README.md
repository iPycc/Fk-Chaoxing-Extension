# Extension Icons

Please add the following icon files to this directory:

- `48.png` - 48x48 pixel icon
- `128.png` - 128x128 pixel icon

These icons will be displayed in Chrome's extension management page.

You can create simple icons using any image editor or use online tools like:
- https://favicon.io/
- https://www.canva.com/

Recommended: Use a simple "CX" text or clipboard icon design.
